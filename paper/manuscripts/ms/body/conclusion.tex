\section{Conclusion} \label{sec:conclusion}

We introduce the $\bob$ approach, which gives a simple method for achieving the fundamental limits on instance optimality in online learning with respect to $\ftl$ and the worst case guarantee of any other input algorithm, drawing on tools from information theory and competitive analysis to do so. It is open whether instance optimal guarantees can be achieved for bandit feedback settings, or whether one can achieve instance optimality with respect to multiple algorithms beyond $\ftl$ and minimax algorithms.
